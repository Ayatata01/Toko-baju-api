# Toko-baju-api
### Cara Penggunaan
1. clone github
2. tambah file .env pada root folder 
3. isikan DB_DATABASE = database mongoDB name, DB_PASSWORD = database password, SECRET = string bebas
4. jalankan perintah npm install
5. jalankan perintah npm start
